# ✋ Hand Sign Detection using CNN

A real-time hand sign detection system built using Convolutional Neural Networks (CNN), OpenCV, and TensorFlow.

---

## 🚀 Features
- Real-time webcam detection
- Supports multiple hand gestures (A-Z, space, delete)
- Custom dataset support
- Lightweight and fast

---

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

---
## 📦 Dataset

This project uses the ASL Alphabet Dataset.

🔗 Download: https://www.kaggle.com/datasets/grassknoted/asl-alphabet

### Setup
- Extract the dataset
- Move folders into:
```
dataset/
train/
test/
```

Ensure folder structure is correct before training.
